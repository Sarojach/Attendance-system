概要

本プロジェクトは、学校における出席管理を簡単かつ効率的に行うことを目的とした、シンプルな出席管理Webアプリケーションです。
サーバーやデータベースを使用せず、HTML、CSS、JavaScript のみで構築されており、ブラウザの LocalStorage を利用してデータを保存・管理します。

主な機能

学生ログイン機能
クラス選択機能
出席登録機能
LocalStorage を使用したデータ保存
ページを再読み込みしてもデータが保持される仕組み
シンプルで分かりやすいUIデザイン

使用技術

HTML：ページ構造の作成
CSS：レイアウトおよびデザイン
JavaScript：ログイン処理、出席管理、データ操作

対応ブラウザ

以下の主要なモダンブラウザで動作確認を行っています。

Google Chrome
Mozilla Firefox
Safari
Microsoft Edge

使用方法

プロジェクトフォルダ内の index.html をWebブラウザで開いてテスト用アカウントのメールアドレスとパスワードを入力してログインします。
ログイン後、出席管理機能を利用できます。
